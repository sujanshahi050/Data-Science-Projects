# PROJECT MOTIVATION

Doing a location based analysis on AirBnb properties near airports.
Used Boston AirBnb open data to carry out the analysis.

Three main questions:
a) How have people rated on locations to those properties closer from the airport?
b) How does average price of AirBnb properties vary as we move farther away from the airport?
c) What does overall ratings of properties look like as we move farther away from airport?
